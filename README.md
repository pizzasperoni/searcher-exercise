# searcher-exercise

Simple exercise using react-redux

### First installed server and client dependencies:


```
npm install
```

and for the client:

```
npm run client-install 
```

### Run the bff and client server using concurrently

```
npm run dev
```

# Tests

```
npm run test
```
and for coverage:

```
npm run test:coverage
```

# Built with

### Client
* React-Redux
* Sass
* Webpack

### Server
* Express
